# Discord TRPG Bot

一個為跑團設計、具備高度自動化與擴充性的 TRPG 輔助系統。

## 核心功能
- **跨伺服器連動**：透過 `include` 機制，讓多個伺服器共享同一個規則卡的屬性定義。
- **沉浸式演繹**：自動偵測玩家註冊的角色，透過 Webhook 以角色身分發言。
- **動態運算引擎**：自定義 AST 解析，支援複雜的擲骰公式與角色屬性連動。
- **熱插拔模組**：支援 `|reload` 指令，無需重啟即可更新邏輯。

## 資料夾結構
- `core/`: 系統核心（資料庫、運算引擎、權限檢查、自動補完）。
- `cogs/`: 功能模組（戰鬥、物品、技能、角色卡、規則定義）。
- `main.py`: 機器人啟動入口。

## 常用指令說明
- `/card set`: 定義伺服器使用的規則卡與屬性。
- `/role add`: 建立角色卡並連結種族。
- `/battle act`: 開啟戰鬥，進行技能或物品操作。
- `/item give`: 分發道具給特定玩家。

## 開發環境
- **語言**: Python 3.12+
- **資料庫**: MongoDB (Motor 驅動)
- **框架**: [discord.py](https://github.com/Rapptz/discord.py)
